# Package index

## All functions

- [`barker_proposal()`](http://github-pages.ucl.ac.uk/rmcmc/reference/barker_proposal.md)
  : Create a new Barker proposal object.

- [`bimodal_barker_proposal()`](http://github-pages.ucl.ac.uk/rmcmc/reference/bimodal_barker_proposal.md)
  : Create a new Barker proposal object with bimodal noise distribution.

- [`chain_state()`](http://github-pages.ucl.ac.uk/rmcmc/reference/chain_state.md)
  : Construct a new chain state.

- [`covariance_shape_adapter()`](http://github-pages.ucl.ac.uk/rmcmc/reference/covariance_shape_adapter.md)
  : Create object to adapt proposal with shape based on estimate of
  target distribution covariance matrix.

- [`dual_averaging_scale_adapter()`](http://github-pages.ucl.ac.uk/rmcmc/reference/dual_averaging_scale_adapter.md)
  : Create object to adapt proposal scale to coerce average acceptance
  rate using dual averaging scheme of Nesterov (2009) and Hoffman and
  Gelman (2014).

- [`example_gaussian_stan_model()`](http://github-pages.ucl.ac.uk/rmcmc/reference/example_gaussian_stan_model.md)
  :

  Construct an example BridgeStan `StanModel` object for a Gaussian
  model.

- [`hamiltonian_proposal()`](http://github-pages.ucl.ac.uk/rmcmc/reference/hamiltonian_proposal.md)
  : Create a new Hamiltonian proposal object.

- [`langevin_proposal()`](http://github-pages.ucl.ac.uk/rmcmc/reference/langevin_proposal.md)
  : Create a new Langevin proposal object.

- [`random_walk_proposal()`](http://github-pages.ucl.ac.uk/rmcmc/reference/random_walk_proposal.md)
  : Create a new (Gaussian) random walk proposal object.

- [`robust_shape_adapter()`](http://github-pages.ucl.ac.uk/rmcmc/reference/robust_shape_adapter.md)
  : Create object to adapt proposal shape (and scale) using robust
  adaptive Metropolis algorithm of Vihola (2012).

- [`sample_chain()`](http://github-pages.ucl.ac.uk/rmcmc/reference/sample_chain.md)
  : Sample a Markov chain

- [`scale_adapter()`](http://github-pages.ucl.ac.uk/rmcmc/reference/scale_adapter.md)
  : Create object to adapt proposal scale to coerce average acceptance
  rate.

- [`shape_adapter()`](http://github-pages.ucl.ac.uk/rmcmc/reference/shape_adapter.md)
  : Create object to adapt proposal shape.

- [`stochastic_approximation_scale_adapter()`](http://github-pages.ucl.ac.uk/rmcmc/reference/stochastic_approximation_scale_adapter.md)
  : Create object to adapt proposal scale to coerce average acceptance
  rate using a Robbins and Monro (1951) scheme.

- [`target_distribution_from_log_density_formula()`](http://github-pages.ucl.ac.uk/rmcmc/reference/target_distribution_from_log_density_formula.md)
  : Construct target distribution from a formula specifying log density.

- [`target_distribution_from_stan_model()`](http://github-pages.ucl.ac.uk/rmcmc/reference/target_distribution_from_stan_model.md)
  :

  Construct target distribution from a BridgeStan `StanModel` object.

- [`variance_shape_adapter()`](http://github-pages.ucl.ac.uk/rmcmc/reference/variance_shape_adapter.md)
  : Create object to adapt proposal with per dimension scales based on
  estimates of target distribution variances.
